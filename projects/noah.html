Noah Transport is a small transport renting company located in Selangor. The website allows visitors to book a vehicle for rent online.

The CSS is fully customized. The front page design is non-scrollable, highlighting the features of the site. The site uses traditional PHP method, no framework is being used.

Unfortunately, the website is already dead and discontinued.
