Installation

Download / fork the package from here

Just attach the source CSS file in the head section.

<link type="text/css" href="path/to/timeline/timeline.min.css" />

Layout and Style

  • To change from vertical to horizontal timeline, just change the class '.timeline-vertical' to '.timeline-horizontal'
  • To change from normal to alternate timeline, just add the class '.timeline-alternate'
  • To set timeline to DARK version, add '.timeline-dark' class to the timeline
  • To add banner image to a timeline item, add
    <div class="timeline-banner"><img src="path/to/image.jpg" /></div>

Color Options

You may set color theme of the timeline based on Bootstrap color classes.

There are also 10 extra color classes that you may use.

Just add any of the following class to the '.timeline' element:

  • '.timeline-primary'
  • '.timeline-secondary'
  • '.timeline-success'
  • '.timeline-danger'
  • '.timeline-warning'
  • '.timeline-info'
  • '.timeline-oak'
  • '.timeline-hazel'
  • '.timeline-forest'
  • '.timeline-golden'
  • '.timeline-scarlet'
  • '.timeline-seafoam'
  • '.timeline-blueberry'
  • '.timeline-turquoise'
  • '.timeline-lavender'
  • '.timeline-fire'
