Salihin Consulting Group is an audit and consultancy firm. They also have a software development division, that most people do not know about. This is where I work, as a senior programmer.​​​​​​​

During my employment there, I was tasked with updating the company website. So I made the website, with custom CMS. The CMS allows the admin to create/edit Post Entry, the entry will be available on the front site once published. The admin can also set the template of the page, upload files and images, customize menu and setup slider. Visitors can also apply jobs online, send message, view events, etc.. Thus, the CMS also allow admin to set all that up, posting up career opportunities, read messages from visitors, setting up events, etc... It is such a huge CMS, with lots of contents.

I use Bootstrap for the skeleton of the styling, with AdminLTE as the main style, then I made a styling extension. The front page design is simple, section-by-section layout. The CMS follows AdminLTE, with a bit of customization.

The engine used is CodeIgniter 3 with MySQL database.

The website is still up and running at http://www.salihin.com.my/
