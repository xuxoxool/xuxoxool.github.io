SKILLX.
A jQuery plugin to chart skill set View Repo on Github

Sample

SKILLX is a jquery plugin, where you can display a set of skills in a visual representation.

The set of skills will be parsed and plotted into a rotary chart, with each hand of the chart corresponds to the value of the skill.

Installation

Download / fork the package from here

Just attach the source JS file. Make sure to put it under jQuery.

<script src="path/to/skillx/js/jquery.js"></script> <script src="path/to/skillx/js/skillx.min.js"></script>

How To Use

Attach the plugin to an element, the element should be a blocked element. You must also specify the element height

$('#skills_1').skillx({ 'maxPoints' : 5, 'showValueAs' : 'percentage', 'flatEdge' : true, 'sort' : { 'enable' : true, 'column' : 'name', 'order' : 'asc' }, 'items' : [ { 'name' : 'PHP', 'value' : 4 }, { 'name' : 'Java', 'value' : 2 }, { 'name' : 'Dot Net', 'value' : 2 }, { 'name' : 'HTML', 'value' : 5 }, { 'name' : 'jQuery', 'value' : 5 }, { 'name' : 'CSS', 'value' : 5 }, { 'name' : 'Bootstrap', 'value' : 3 }, { 'name' : 'SASS/LESS', 'value' : 1 }, { 'name' : 'Angular', 'value' : 1 }, { 'name' : 'React', 'value' : 1 }, { 'name' : 'Xuxo', 'value' : 5 }, { 'name' : 'Code Igniter', 'value' : 4 }, { 'name' : 'Zend', 'value' : 3 }, { 'name' : 'Wordpress', 'value' : 3 }, { 'name' : 'Laravel', 'value' : 1 }, { 'name' : 'Malay', 'value' : 5 }, { 'name' : 'English', 'value' : 4 } ], 'onItemClick' : function(a) { alert(a.name + ' : ' + a.value); } });

Properties

Property Description Values Default
'maxPoints' Maximum points in the skill set Integer number starting from 1 5
'showValueAs' To show value as percentage or number 'percentage', 'number' 'percentage'
'flatEdge' To show the edge of each skill item as flat or sharp true, false false
'sort' Sorting options for skill set items
{ 'enable' : [true/false] 'column' : Column name such as 'value' / 'name' / or any specified (e.g: 'id', 'date') 'order' : ['asc'/'desc'] }
'items' Skill set items should follow this format:
{ 'name' : 'Item Name' ([Required] - Name of the item, string) 'value' : 3 ([Required] - Integer value of the item, should not be more than maxPoints) 'bgColor' : '#FF0000' ([Optional] - Hex code for the color of the item background) 'textColor' : '#222222' ([Optional] - Hex code for the color of the item text) 'class' : 'foo bar' ([Optional] - Additional class for each item, separated by a space) 'attr' : { 'attrName' : 'attrValue', 'attrName' : 'attrValue', } ([Optional] Additional attributes to be passed into each item) 'data' : { 'dataKey' : 'dataValue', 'dataKey' : 'dataValue', } ([Optional] Additional data to be passed into each item) }

Example Property

In this example, the items will be sorted ascending by 'id' column, with sharp edge, and value shown as percentage (See sample no 3)

$('#skills_3').skillx({ 'maxPoints' : 5, 'showValueAs' : 'percentage', 'flatEdge' : false, 'sort' : { 'enable' : true, 'column' : 'id', 'order' : 'asc' }, 'items' : [ { 'name' : 'January', 'value' : 4, 'bgColor' : '#FF0000', 'textColor' : '#222222', 'id' : 1 }, { 'name' : 'February', 'value' : 2, 'id' : 2 }, { 'name' : 'March', 'value' : 2, 'id' : 3 }, { 'name' : 'April', 'value' : 5, 'id' : 4 }, { 'name' : 'May', 'value' : 5, 'id' : 5 }, { 'name' : 'June', 'value' : 5, 'id' : 6 }, { 'name' : 'July', 'value' : 3, 'id' : 7 }, { 'name' : 'August', 'value' : 1, 'id' : 8 }, { 'name' : 'September', 'value' : 1, 'id' : 9 }, { 'name' : 'October', 'value' : 1, 'id' : 10 }, { 'name' : 'November', 'value' : 5, 'id' : 11 }, { 'name' : 'December', 'value' : 4, 'id' : 12 } ], 'onItemClick' : function(a) { alert(a.name + ' : ' + a.value); } });
