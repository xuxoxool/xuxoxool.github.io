SPS E-Voting is a cloud-based, secured e-voting platform for any organizations who want to conduct an election. The election admin will have to pay for subscription, and manage their voting online.​​​​​​​

During my employment at Salihin Consulting Group, I was tasked with developing the E-Voting System. The system allows user to subscribe for an election project, setup their election details such as venue, date and time, candidates and list of voters. Voters then can vote online via the link provided by the election admin.

I use Bootstrap for the skeleton of the styling, with AdminLTE as the main style, then I made a styling extension. The front page design is simple, section-by-section layout. The User Dashboard follows AdminLTE, with a bit of customization.

The engine used is CodeIgniter 3 with MySQL database. Payment Gateway is from BillPlz.

The website is still up and running at http://www.salihinpremier.com/evoting
